## How to Load 1v1freezetag.expblocks to VEX EXP Code

Follow these steps to import the 1v1freezetag.expblocks file into VEX EXP:

### Step 1: Navigate to VEX Code EXP
Open your web browser and go to [VEX Code EXP](https://codeexp.vex.com/).

### Step 2: Create or Open a Project
- Click on **"New Project"** to create a new project, or open an existing one
- Choose your robot type (3D Robot or Basebot) if creating a new project

### Step 3: Import the Block File
1. Click on the **"File"** menu in the top navigation
2. Select **"Import Blocks"** or **"Upload Blocks"** (depending on your interface version)
3. Browse your computer and select the `1v1freezetag.expblocks` file from this repository
4. Click **"Open"** to import the blocks

### Step 4: Verify Import
- The custom blocks from the 1v1freezetag.expblocks file should now appear in your blocks palette
- Look for a category labeled with the blocks from this project (typically organized by function)
- The blocks are now ready to use in your program

### Step 5: Build Your Program
- Drag and drop the imported blocks into your main workspace
- Connect them with other VEX EXP blocks to build your complete 1v1 Freeze Tag program
- Test your code on your robot or in the simulator

### Troubleshooting
- **Blocks not appearing**: Refresh the page (F5) or reload VEX Code EXP
- **Import fails**: Ensure the file is a valid `.expblocks` file and hasn't been corrupted
- **Block compatibility**: Make sure your VEX Code EXP is up to date for best compatibility
